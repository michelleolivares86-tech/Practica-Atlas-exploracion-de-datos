## ¿Qué es Atlas?

**Atlas es el Proyecto 1 del itinerario de Data Analyst.** Es tu primer análisis de datos real.

Aquí harás un análisis descriptivo completo usando **Google Sheets**: abrirás datos reales, harás cálculos sencillos (conteos, sumas, promedios), crearás gráficos básicos e interpretarás los resultados.

No vamos a limpiar datos complejos, construir modelos avanzados ni hacer predicciones. Pero sí vas a **hacer cosas reales** con datos de verdad, con tus propias manos.

Este es el primer paso práctico del bootcamp, y probablemente el más importante: trabajar con datos reales desde el primer momento.

**¿Por qué Google Sheets en este proyecto?**
- Es visual y accesible (no necesitas instalar nada)
- Puedes compartir fácilmente tu trabajo
- Te permite concentrarte en **entender los datos** sin preocuparte de sintaxis de programación
- Las habilidades analíticas que desarrollas aquí son transferibles a cualquier herramienta (Python, R, SQL...)

## 📚 Qué aprenderás en este satélite

En Atlas desarrollarás una habilidad fundamental: **el análisis descriptivo básico de datos**.

| **Habilidad** | **Descripción** |
|---------------|------------------|
| **Lectura de datos** | Abrir y explorar archivos de datos en Google Sheets |
| **Cálculos descriptivos** | Contar filas, sumar valores, calcular promedios y porcentajes |
| **Visualización básica** | Crear gráficos de barras, líneas y sectores para identificar patrones |
| **Formulación de preguntas** | Hacer las preguntas correctas ante un dataset nuevo |
| **Detección de patrones** | Identificar tendencias, valores atípicos y características relevantes |
| **Interpretación crítica** | Entender qué cuentan los números y qué limitaciones tienen |
| **Intuición analítica** | Desarrollar criterio sobre el comportamiento de datos reales |

Todo esto **haciendo**, no solo leyendo. Trabajarás con datos reales en una hoja de cálculo, viendo cada paso que das. Sin asumir que los datos están perfectos. De hecho, aprenderás que **los datos reales nunca están perfectos**, y que justamente eso es lo interesante.
